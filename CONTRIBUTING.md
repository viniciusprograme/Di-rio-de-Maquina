# 🤝 Guia de Contribuição

Obrigado por se interessar em contribuir para o **Diário de Bordo — Máquinas**!

## 📋 Como Contribuir

### 1. Fork o Repositório
```bash
git clone https://github.com/viniciusprograme/Di-rio-de-Maquina.git
cd Di-rio-de-Maquina
```

### 2. Crie uma Branch para sua Feature
```bash
git checkout -b feature/sua-feature
# ou para correções:
git checkout -b fix/seu-fix
```

### 3. Faça suas Mudanças
- Mantenha o código limpo e bem documentado
- Siga o padrão de código existente
- Adicione comentários quando necessário

### 4. Commit com Mensagens Claras
```bash
# Feature
git commit -m "feat: descrição da funcionalidade"

# Correção
git commit -m "fix: descrição do problema corrigido"

# Documentação
git commit -m "docs: descrição da mudança"
```

### 5. Push e Crie um Pull Request
```bash
git push origin feature/sua-feature
```

Depois abra um PR no GitHub com descrição clara do que foi mudado.
